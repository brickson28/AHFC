# Annette's Heart Foundation

Static website for Annette's Heart Foundation for Children.

## Repository structure

Keep these files together in the repository root:

```text
index.html
vercel.json
annettes-heart-logo.png
stethoscope-heart-hero.png
annette-hospital-care.jpeg
annette-after-surgery-2007.jpeg
```

## Deploy with Vercel

1. Create a GitHub repository and upload all five website files.
2. In Vercel, choose **Add New → Project**.
3. Import the GitHub repository.
4. Leave **Framework Preset** set to **Other**.
5. Make sure **Root Directory** is the folder that directly contains `index.html`
   and `vercel.json`.
6. Leave **Build Command** and **Output Directory** empty, then deploy.

The website uses plain HTML, CSS, and JavaScript, so it does not require a build step or external packages.
